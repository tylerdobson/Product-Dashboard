# Product Dashboard (Vanilla JavaScript)

  A simple product dashboard that fetches product data from an external API and displays it as
  interactive cards. Built with plain HTML, CSS, and JavaScript for **Coding Project 02
  (ISM3232)**.

  ## Features
  - Fetches live product data from the Store Products API
  - Two fetch implementations: one using Promises (.then() / .catch()) and one using
  async/await with try/catch
  - Renders the first 5 products with name, image, and price
  - Reusable error handler for failed requests
  - Hover effect on cards (smooth scale + shadow)

  ## Files
  - index.html — page layout (header + product container)
  - style.css — card styling and hover effects
  - cp_2.js — fetch logic, product rendering, and error handling

  ## API
  Data is pulled from:
  https://www.course-api.com/javascript-store-products
